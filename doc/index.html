二、用户/玩家协议

1. 账户与使用

  • 使用本平台需遵守法律法规与社区规则；未成年人应在监护人指导下使用。
  • 若涉及账号体系，用户应妥善保管凭证；因用户保管不善导致的风险由用户自行承担。

2. 服务与限制

  • 平台提供服务器展示、检索/筛选、讨论区与公告等功能；具体功能以平台上线为准。
  • 平台可因合规、技术或运营需要调整、暂停或终止部分功能，并于合理范围内公告。

3. 用户行为规范

  • 不得发布违法、侵权、低俗、诈骗、恶意代码或误导性内容；不得进行骚扰、刷屏或干扰平台运行。
  • 尊重他人权利与社区氛围；违反者将面临内容删除、禁言、封禁或移交监管等处理。

4. UGC 内容与责任

  • 服务器卡片、介绍、图片视频、外链等由提交者提供，平台仅作展示。
  • 用户对其发布内容独立负责，平台不对第三方内容的真实性、完整性与可用性作保证。

5. 免责与风险提示

  • 平台按“现状/可获得”提供，对网络波动、第三方故障、不可抗力等引发的损失不承担赔偿责任。

三、服主协议（服务器接入条款）

1. 准入与真实性

  • 提交者应为服主/管理员或经授权代表；提交信息需真实、准确、可验证。
  • 不得包含恶意软件、钓鱼链接、侵权素材或违反公序良俗的内容。

2. 展示与排序

  • 平台可基于质量、活跃度、合规性与用户反馈进行排序与推荐；具体策略可能随业务优化调整。
  • 对违规或不适宜的服务器条目，平台有权下架、限流或永久移除。

3. 知识产权与授权

  • 提交即授予平台在全球范围内非独占、免版税、可撤销的使用、复制、存储、展示与传播许可，仅用于运营与推广本平台。
  • 服主须确保素材（含图文与视频/音频、商标）已获合法授权，不侵害第三方权益。

4. 监测与合规

  • 平台可进行合规审核与异常监测；对多次投诉或涉嫌违法的服务器将限制展示或移交监管。
  • 服主应配合处理用户投诉、权利主张与安全事件，及时更新失效/变更信息。

四、第三方合作协议（摘要）

  • 合作范围：品牌共建、活动联办、资源互换、技术接口与统计工具等；以书面协议为准。
  • 合规要求：遵守数据与隐私、广告合规与未成年人保护要求；不得夹带诱导、违规营销。
  • 数据原则：最小必要、目的限定、授权透明；未经许可不得擅自二次分发或画像。
  • 责任边界：各方对自身内容/服务/合规各自负责；联合活动另行约定保险/责任分担。

注：正式合作以双方签署的《合作协议》《数据处理协议（DPA）》为准，本节为平台通用条款摘要。

五、隐私政策（含涉税信息报送）

1. 收集信息类型

  • 必要信息：提交人联系方式、服务器元信息、审核材料。
  • 设备与日志：设备标识、浏览器类型、IP、时间戳、操作日志、异常记录、Cookie/本地存储。
  • 合规模块：合规标签、风险判定摘要、审计流水号。

2. 使用目的

  • 提供/优化展示与检索、风控与安全、问题定位与统计。
  • 符合法律法规的监管报送、纠纷处理与权利保护。

3. 涉税信息报送

  • 依据税收征管要求，可能通过电子税务局官方端口报送涉及交易/结算/激励的必要信息。
  • 传输全程加密（如 TLS），配合端口所需报文签名、证书校验、时间戳与完整性校验。
  • 留存周期与字段范围以法律法规与监管口径为准，逾期删除或匿名化处理。

4. 共享与披露

  • 不出售个人信息；仅在取得授权、依法依规、或为维护重大权益时共享。
  • 第三方合作遵循最小必要与目的限定，需签署 DPA 并接受审计。

5. 安全措施

  • 分级分域存储、最小权限（RBAC）、MFA、密钥分离、加密存储与细粒度审计。
  • 备份与恢复演练、漏洞管理与供应链依赖巡检；重大事件依法通知用户与主管部门。

6. 用户权利

  • 访问、更正、删除、撤回同意、获取说明与账号注销；依法必须留存的数据不适用强制删除。
  • 未成年人：建议由监护人代为行使相关权利与监督使用。

六、知识产权与侵权处理

  • 平台与标识受版权与商标法保护；未经许可不得使用。
  • UGC 版权归属原作者；提交即授予平台在运营与推广范围内的非独占使用权。
  • 侵权投诉应提供权利证明、侵权链接、联系方式与处理诉求；经核实将尽快处理。

八、如何添加服务器/展示

  1. 准备信息：名称、封面（可选）、服务器地址与端口、版本/模组、标签、简介、外链（官网/视频/企鹅群）。
  2. 提交表单：在 mcmsf.com/submit 填写信息并上传素材。
  3. 合规校验：完成基础校验（恶意链接/侵权/敏感内容），必要时补充授权与证明。
  4. 审核与上线：通常 T+3 工作日内完成审核；结果将通过邮箱反馈。
  5. 维护与更新：版本变更、地址调整或重大活动，请在 /dashboard 同步更新。

提示：请避免夸大或虚假宣传；若服务器长期不可达或屡遭投诉，可能被降权或下架。

  • 争议解决：先行协商；不成的，提交平台所在地有管辖权的人民法院。
  • 责任上限：除法律强制规定外，对直接损失的总体赔偿责任以合理且必要的限额为准。
